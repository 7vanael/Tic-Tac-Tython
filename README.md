# Tic-Tac-Tython

This is a game of Tic-Tac-Toe in Python where you can 
play against an unbeatable computer opponent. Good luck!


### Install dependencies: 

Download and install [Python](https://wiki.python.org/moin/BeginnersGuide/Download).

`pip3 install -r requirements.txt`

### Run

To play: `python3 -m main`

### Development


#### Running tests
`pytest -v` or `pytest filename.py -v`

If you have changed the dependencies, run
`pip freeze > requirements.txt`
to maintain the dependency list for other users.
